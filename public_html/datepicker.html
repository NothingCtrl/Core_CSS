Date picker

Core_CSS provides datepicker component for getting/setting date values. 

        <div data-role="datepicker"></div>





Options

Option Data-* Default Description
mode data-mode default Show datepicker in specific mode: default, multi
locale data-locale CORE_LOCALE Show datepicker in specific language. For more info see locales
toolbar data-toolbar true Show datepicker toolbar
footer data-footer true Show datepicker footer
preset data-preset [ ] Preset values
current data-current null Current values
buttons data-buttons ['cancel', 'today', 'clear', 'done'] Show specific datepicker footer buttons
onCreate data-on-create $.noop Callback for on create event
onDone data-on-done $.noop Callback for on done event
onCancel data-on-cancel $.noop Callback for on cancel event
onToday data-on-today $.noop Callback for on today event
onClear data-on-clear $.noop Callback for on clear event
onDay data-on-day $.noop Callback for on day event

Dialog mode

Core_CSS provides method to create date picker widget as dialog. To create date picker dialog you can call corePickers.datePicker method. For more information about corePickers object see pickers docs. 

        corePicker.datePicker(function(val){
            ... callback for onDone ...
        }, {
            ... options ...
        })
