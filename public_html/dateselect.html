Date select

Core_CSS provides components for getting/setting date and time values. 

        <div data-role="dateselect"></div>

Options

Option Data-* Default Description
locale data-locale CORE_LOCALE Show datepicker in specific language. For more info see locales
minYear data-min-year 1900 Min value for year
maxYear data-max-year Current year Maxvalue for year
day data-day Current day Current selected Day value
month data-month Current day Current selected Month value
year data-year Current year Current selected Year value
buttons data-buttons ['cancel', 'random', 'today', 'done'] Show specific datepicker footer buttons
onDone data-on-done $.noop Callback for on done event

Dialog mode

Core_CSS provides method to create date select widget as dialog. To create date select dialog you can call corePickers.dateSelect method. For more information about corePickers object see pickers docs. 

        corePicker.dateSelect(function(val){
            ... callback for onDone ...
        }, {
            ... options ...
        })
