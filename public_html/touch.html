Touch & Swipe

Core_CSS contains widget for handling touch and swipe. This widget based on TouchSwipe from Matt Bryson.

Widget can be used on touch devices such as iPad, iPhone, Android etc. Detects single and multiple finger swipes, pinches and falls back to mouse 'drags' on the desktop. Time and distance thresholds can be set to distinguish between swipe gesture and slow drag. Allows exclusion of child elements (interactive elements) as well allowing page scrolling or page zooming depending on configuration.

Widget features:

  • Detects swipes in 4 directions, "up", "down", "left" and "right"
  • Detects pinches "in" and "out"
  • Supports single finger or double finger touch events
  • Supports click events both on the touchSwipe object and its child objects
  • Definable threshold / maxTimeThreshold to determin when a gesture is actually a swipe
  • Events triggered for swipe "start","move","end" and "cancel"
  • End event can be triggered either on touch release, or as soon as threshold is met
  • Allows swiping and page scrolling
  • Disables user input elements (Button, form, text etc) from triggering swipes

Usage

        $(function() {
            $("#test").touch( {
                swipe:function(event, direction, distance, duration, fingerCount, fingerData) {
                  $(this).text("You swiped " + direction );
                }
            });
        });

Options

Option Data-* Default Description
fingers data-fingers 1 Constants representing the number of fingers used in a swipe.
threshold data-threshold 75 By setting the threshold you can set how far the user must swipe before it is considered a swipe.
cancelThreshold data-cancel-threshold null
pinchThreshold data-pinch-threshold 20
maxTimeThreshold data-max-time-threshold null
fingerReleaseThreshold data-finger-release-threshold 250
longTapThreshold data-long-tap-threshold 500
doubleTapThreshold data-double-tap-threshold 200
swipe data-swipe null Function. Params: event, direction, distance, duration, fingerCount, fingerData, currentDirection
swipeLeft data-swipe-left null Function. Params: event, direction, distance, duration, fingerCount, fingerData, currentDirection
swipeRight data-swipe-right null Function. Params: event, direction, distance, duration, fingerCount, fingerData, currentDirection
swipeUp data-swipe-up null Function. Params: event, direction, distance, duration, fingerCount, fingerData, currentDirection
swipeDown data-swipe-down null Function. Params: event, direction, distance, duration, fingerCount, fingerData, currentDirection
swipeStatus data-swipe-status null Function. Params: event, phase, direction, distance, duration, fingerCount, fingerData, currentDirection
pinchIn data-pinch-in null Function. Params: event, direction, distance, duration, fingerCount, zoom, fingerData
pinchOut data-pinch-out null Function. Params: event, direction, distance, duration, fingerCount, zoom, fingerData
pinchStatus data-pinch-status null Function. Params: event, direction, distance, duration, fingerCount, zoom, fingerData
tap data-tap null Function. Params: event, target
doubleTap data-double-tap null Function. Params: event, target
longTap data-long-tap null Function. Params: event, target
hold data-hold null Function. Params: event, target
triggerOnTouchEnd data-trigger-on-touch-end true With triggerOnTouchEnd you can trigger the swipe end handler either when the user releases (default) or when the user has swiped the distance / time of the thresholds (but is still swiping).
triggerOnTouchLeave data-trigger-on-touch-leave false With triggerOnTouchLeave you can end the event if the user swipes off the element
allowPageScroll data-allow-page-scroll auto Allow or disallow page scroll
fallbackToMouseEvents data-fallback-to-mouse-events true Generate fallback to mouse events
excludedElements data-excluded-elements .noSwipe If you want to exclude certain child elements from triggering swipes, you can simply add a .noSwipe class to the element. Then the element and any of its children will no longer trigger the swipe. Also, a jQuery selector is used to exclude input elements as well as the .noSwipe elements. By default all button, input, select, textarea and a elements are excluded. So either add a .noSwipe class the element, or set your own selector in the excludedElements property.
preventDefaultEvents data-prevent-default-events true Call preventDefault on events

Methods

Method Params Description
enable Enable touch and swipe
disable Disable touch and swipe
