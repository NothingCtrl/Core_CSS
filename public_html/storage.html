Storage engine

Core_CSS provides special object to work with browser storage engine. This object named as coreStorage and contains next methods:

  • coreStorage.setKey(key) - setup global storage key for app
  • coreStorage.setItem(key, value) - store any values with key
  • coreStorage.getItem(key, default, reviver) - get stored values with key. You get default if key not founded.
  • coreStorage.getItemPart(key, sub_key, default, reviver) - get part of stored values with key. You get default if key not founded.
  • coreStorage.delItem(key) - Remove value from storage for specified key.
