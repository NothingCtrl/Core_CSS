Time picker

Core_CSS provides datepicker component for getting/setting time values. 

        <div data-role="timepicker"></div>



Options

Option Data-* Default Description
locale data-locale CORE_LOCALE Show picker in specific language. For more info see locales
time data-time Current time Init time, string or datetime object
onDone data-on-done $.noop Callback for on done event
onChange data-on-change $.noop Callback for on change event
onCancel data-on-cancel $.noop Callback for on cancel event

Dialog mode

Core_CSS provides method to create date picker widget as dialog. To create date picker dialog you can call corePickers.timePicker method. For more information about corePickers object see pickers docs. 

        corePickers.timePicker(function(val){
            ... callback for onDone ...
        }, {
            ... options ...
        })
