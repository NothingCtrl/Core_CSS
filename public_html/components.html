Core_Components

Core_CSS includes a many widgets (Core_Components) for creating cool interfaces in Material Design Style. All components are written with Google Material Design guidelines.

20+

Some components may require activation via data attribute data-role with value of component name. 

        <ul data-role="accordion">
            ...
        </ul>

Some components have any options. You can use the options by defining them through data attributes data-*. 

        <ul class="accordion"
            data-close-other="false"
            data-on-open="_function_name_"
            data-on-close="console.log(arguments)"
            data-role="accordion">
        ...
        </ul>
