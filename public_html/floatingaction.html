Floating action button

Floating action buttons are used for a promoted action. They are distinguished by a circled icon floating above the UI and have motion behaviors that include morphing, launching, and a transferring anchor point.

Floating action buttons come in two sizes:

  • Default size: For most use cases
  • Mini size: Only used to create visual continuity with other screen elements
        <button class="action-button floating-action-bottom">...</button>
        <button class="action-button mini floating-action-top">...</button>

Multi actions

Core_CSS provides simple method for create multi action button.

Core_CSS 
        <div class="multi-action floating-action-bottom">
            <button class="action-button bg-red fg-white"
                    onclick="$(this).toggleClass('active')">
                <span class="i-add icon"></span>
            </button>
            <ul class="actions">
                <li class="bg-blue"><a href="#"><span class="i-user_add"></span></a></li>
                <li class="bg-teal"><a href="#"><span class="i-library"></span></a></li>
                <li class="bg-pink"><a href="#"><span class="i-alarm"></span></a></li>
                <li class="bg-orange"><a href="#"><span class="i-lock"></span></a></li>
            </ul>
        </div>
