Lists

Lists present multiple line items vertically as a single continuous element. Core_CSS provides three predefined list styles: list-item, feed-item and list-item-custom, 

        <ul class="list">
            <li class="list-item">...</li>
            ...
            <li class="list-item">...</li>
        </ul>
        ...
        <ul class="list">
            <li class="feed-item">...</li>
            ...
            <li class="feed-item">...</li>
        </ul>
        ...
        <ul class="list">
            <li class="list-item-custom">...</li>
            ...
            <li class="list-item-custom">...</li>
        </ul>
