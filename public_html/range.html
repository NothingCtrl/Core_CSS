Range

Range let users select values by moving the slider thumb. Core_CSS provides simple method to create range element.

Horizontal




        <div data-role="range" data-position="35"></div>

Vertical


        <div data-role="range" data-vertical="true" data-position="35"></div>

Options

Option Data-* Default Description
vertical data-vertical false Show range in vertical orientation
position data-position 0 Init range position
buffer data-buffer 0 Buffer size (0-100) in percent for slider (great for players)
accuracy data-accuracy 0 Range accuracy
showHint data-show-hint false Show or hide hint for value
permanentHint data-permanent-hint false Show hint for value in permanent mode
hintPosition data-hint-position top | right Hint position: top, bottom or left, right
color data-color default Range back color
completeColor data-complete-color default Range complete color
bufferColor data-buffer-color default Range buffer color
markerColor data-marker-color default Range marker color
colors data-colors false Defines a set of colors that will be applied uniformly to complete color when marker is moving. String with comma delimiter.
minValue data-min-value 0 Range min value
maxValue data-max-value 100 Range max value
returnType data-return-type value Return value as value or percents
target data-target false Bound element
onStartChange data-on-start-change $.noop Callback
onChange data-on-change $.noop(val) Callback
onChanged data-on-changed $.noop(val) Callback
onBufferChange data-on-buffer-change $.noop(val) Callback

Methods

Method Params Description
value val or nothing Set or get range value
buffer val or nothing Set or get range buffer value
