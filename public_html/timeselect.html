Time select

Core_CSS provides components for getting/setting time values. 

        <div data-role="timeselect"></div>

Options

Option Data-* Default Description
locale data-locale CORE_LOCALE Show picker in specific language. For more info see locales
hour data-hour 0 Current hour value
minute data-minute 0 Current minute value
second data-second 0 Current second value
buttons data-buttons ['cancel', 'random', 'now', 'done'] Show specific picker footer buttons
onDone data-on-done $.noop Callback for on done event

Dialog mode

Core_CSS provides method to create date select widget as dialog. To create date select dialog you can call corePickers.timeSelect method. For more information about corePickers object see pickers docs. 

        corePickers.timeSelect(function(val){
            ... callback for onDone ...
        }, {
            ... options ...
        })
