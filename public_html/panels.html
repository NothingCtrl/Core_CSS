Expansion panels

Core_CSS provides special classes and widget to create expansion panels.

Static panel


Header
The thing invents enlightenment which is not powerful. Chicken breasts taste best with whiskey and lots of szechuan pepper.

Expansion panel

Options

Option Data-* Default Description
collapseMode data-collapse-mode full Hide footer on collapse
duration data-duration 300 Collapse speed
open data-open true Default mode for panel: opened or closed

Methods

Method Params Description
open Expand panel
close Collapse panel
toggle Toggle state of panel
isOpened Toggle state of panel
setContent html SSet panel content

Object

Core_CSS provides corePanels object to panels manipulate. corePanels contains next methods:

  • corePanels.isPanel(el) - check if object is panel
  • corePanels.open(el) - open required panel
  • corePanels.close(el) - close required panel
  • corePanels.toggle(el) - toggle state of required panel
