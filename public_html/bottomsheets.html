Bottom sheets

Bottom sheets slide up from the bottom of the screen to reveal more content. Bottom sheets are displayed only as a result of a user-initiated action.

Modal bottom sheets may present actions in a list or grid as an alternative to menus or simple dialogs.

Options

Property Data-* Default Description
mode data-mode list Display mode grid or list
overlay data-overlay false Display overlay true or false
duration data-duration 300 Display speed

Methods

Name Description
open Open sheet
close Close sheet
toggle Toggle sheet 
        var el = $('#el_id').data('bottomsheet');
        el.open('list');
        el.open('grid');
        el.close();
        el.toggle('list');
        el.toggle('grid');

Global object

Core_CSS provides a global object coreBottomSheet to manipulate of Element.

Name Description
coreBottomSheet.open Open sheet
coreBottomSheet.close Close sheet
coreBottomSheet.toggle Toggle sheet 
        coreBottomSheet.open(element, mode);
        coreBottomSheet.toggle(element, mode);
        coreBottomSheet.close(element);
