Gravatar helper

Core_CSS contains method to get your Gravatar image.

Getting gravatar

        <img data-role="gravatar" data-email="sergey@pimenov.com.ua">
        <div data-role="gravatar" data-email="sergey@pimenov.com.ua"></div>

        <img id="avatar_1">
        $("#avatar_1").attr("src", coreGravatars.getImageSrc('sergey@pimenov.com.ua'));

        <div id="avatar_2"></div>
        $("#avatar_2").html(coreGravatars.getImage('sergey@pimenov.com.ua'));

Any size

        <img data-role="gravatar" data-email="sergey@pimenov.com.ua" data-size="40">
        <img data-role="gravatar" data-email="sergey@pimenov.com.ua" data-size="80">
        <img data-role="gravatar" data-email="sergey@pimenov.com.ua" data-size="120">

Object

coreGravatars object contains two methods:

  • coreGravatars.getImage - getting image element
  • coreGravatars.getImageSrc - getting image source path

coreGravatars.getImage

        coreGravatars.getImage(email, size, def, is_jquery_object);

coreGravatars.getImageSrc

        coreGravatars.getImageSrc(email, size, def);

Parameters

Parameter Description
email Email for which you want to image
size Size of result image
def Default wrapper for result image if not found
is_jquery_object Return type for getImage method: element or jQuery object
