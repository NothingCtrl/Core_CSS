Quick start

This is a page template for app with Core_CSS. 

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport"
                  content="user-scalable=no,
                  initial-scale=1,
                  maximum-scale=1,
                  minimum-scale=1,
                  width=device-width" />
            <meta http-equiv="Content-Security-Policy"
                  content="...your_security_definition..." />

            <link rel="stylesheet" type="text/css" href="css/core.min.css">
            <link rel="stylesheet" type="text/css" href="css/core-colors.min.css">
            <link rel="stylesheet" type="text/css" href="css/core-icons.min.css">

            <title>App template</title>
        </head>
        <body>
            <div class="app">
                <header class="header">...</header>
                <div class="sidebar" data-role="sidebar">...</div>
                <div class="bottomnav">...</div>
                <ul class="bottomsheet">...</ul>

                <div class="view-content">
                    ...your_pages...
                </div>
            </div>

            <script src="js/jquery-3.1.1.min.js"></script>
            <script src="js/core.min.js"></script>
            <script src="js/app.js"></script>
        </body>
        </html>

Components initialising

All components can be initialized over data-role attribute or calling jQuery function.

If you init components over dat-role you must set component options in data-* attributes.

Example

        <div data-role="timepicker" data-locale="uk-UA"></div>

Component methods

To call components methods you must get component object and them call required method. 

        <div data-role="dialog" id="dialog1">
            ...
        </div>

        <script>
            var html = "dialog new content";
            var dlg = $("#dialog1").data("dialog");
            dlg.setContent(html);
            dlg.open();
        </script>
