Tabs

Tabs make it easy to explore and switch between different views. Tabs enable content organization at a high level, such as switching between views, data sets, or functional aspects of an app. 

        <ul data-role="tabs">
            <li class="tab ">One</li>
            <li class="tab ">Two Item</li>
            <li class="tab ">Three</li>
        </ul>

Options

Option Data-* Default Description
target data-target self The element that contains the frames. Default in same level with tabs
tabsColor data-tabs-color bg-lightBlue Tabs background color. Class name or hex value.
fontColor data-font-color fg-white Tabs font color. Class name or hex value.
markerColor data-marker-color bg-yellow Tabs marker color. Class name or hex value.
deep data-deep normal In normal mode tabs has z-index as header. Can be: normal or deep
onTab data-on-tab $.noop Callback for tab change.

Tabs target

        <ul data-role="tabs" data-target="#element_id">
            <li class="tab" data-target="gallery">Gallery</li>
            <li class="tab" data-target="description">Description</li>
            <li class="tab" data-target="feed">Feed</li>
        </ul>
        <div id="element_id">
            <div id="gallery">...</div>
            <div id="description">...</div>
            <div id="feed">...</div>
        </div>

    Tabs state

    To disable tab you can add class disabled to required tab. 

            <ul data-role="tabs">
            <li class="tab ">One</li>
            <li class="tab disabled">Two Item</li>
            <li class="tab ">Three</li>
        </ul>
