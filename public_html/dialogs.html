Dialogs

Dialogs inform users about a specific task and may contain critical information, require decisions, or involve multiple tasks.

Dialogs contain text and UI controls. They retain focus until dismissed or a required action has been taken. Use dialogs sparingly because they are interruptive.

Core_CSS provides two ways for create dialogs:

  1. Predefined method
  2. Runtime method

To use first method you must create div with attribute data-role="dialog" and them call dialog methods. 

        <div data-role="dialog" id="d1">
            ...
        </div>

        <script>
            var d = $("d1").data('dialog');
            d.open();
        </script>

For second method you must use $.Dialog or coreDialog objects. 

        <script>
            $.Dialog({
                ...dialog_options...
            });

            or

            coreDialog.create({
                ...dialog_options...
            });
        </script>

Options

Objects

coreDialog

Core_CSS provides object coreDialog to create and manage dialogs. This object contains next methods:

  • coreDialog.open(el, content, contentType)
  • coreDialog.close(el)
  • coreDialog.toggle(el, content, contentType)
  • coreDialog.create(data)

$.Dialog

$.Dialog is a synonym to coreDialog.create(...).

Data for create dialog method

Data can contains: title, content, actions and dialog options.

Examples

Predefined dialog

        <button class="raised-button" onclick="coreDialog.open('#dialog1')">Open dialog</button>

        <div id="dialog1" data-role="dialog" data-overlay="true">
            <div class="dialog-title">
                Use Google's location service?
            </div>
            <div class="dialog-content">
                Bassus abactors ducunt ad triticum.
                A fraternal form of manifestation is the bliss.
            </div>
            <div class="dialog-actions">
                <button class="flat-button" onclick="coreDialog.close('#dialog1')">disagree</button>
                <button class="flat-button" onclick="coreDialog.close('#dialog1')">agree</button>
            </div>
        </div>

Runtime dialog

        $.Dialog({
            title: "Dialog title",
            content: "This is a dialog content",
            actions: [
                {
                    title: "Ok",
                    onclick: function(el){
                        alert("You pressed OK button");
                    }
                },
                {
                    title: "Cancel",
                    cls: "js-dialog-close"
                }
            ],
            options: {
                type: "warning"
            }
        });
