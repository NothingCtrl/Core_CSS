Donut chart

Core_CSS provides simple method to create donut chart. 

        <div data-role="donut" ...></div>

Options

Option Data-* Default Description
size data-size 100 Donut size in pixels
radius data-radius 50 Donut radius in pixels
value data-value 0 Donut value
backgrouns data-background #ffffff Donut background color in hex
stroke data-stroke #d1d8e7 Donut stroke color in hex
fill data-fill #49649f Donut fill color in hex
color data-color #49649f Donut text color in hex
fontSize data-font-size auto Donut title font size in pixels or auto size
hole data-hole 0.8 Donut hole size
total data-total 100 Donut total value
cap data-cap % Donut title sign (only text)
animate data-animate 0 Donut animate speed
showTitle data-show-title true Donut show/hide title

Methods

Method Params Description
val value New donut value 
        <div id="donut1" data-role="donut" ...></div>

        <script>
            var new_value = coreUtils.random(0,100);
            var d = $("#donut1").data("donut");

            d.val(new_value);
        </script>
