Steppers

Core_CSS contains classes for creating steppers. 

        <div data-role="stepper" ...></div>
    

        <script>
            var steps_func = function(){
                return ['F1', 'F2', 'F3', 'F4'];
            }
        </script>

        <div data-role="stepper" data-steps="4" class="margin2" data-clickable="false"></div>
        <div data-role="stepper" data-steps="4" data-label="Step" class="margin2"></div>
        <div data-role="stepper" data-steps='["Step1", "Step2", "Step3"]' class="margin2"></div>
        <div data-role="stepper" data-steps='["Start", "", "", "Finish"]' class="margin2"></div>
        <div data-role="stepper" data-steps="steps_func" data-start-step="2" class="margin2"></div>
        <div data-role="stepper" data-steps="steps_func" data-states='["current", "complete", "warning", "error"]' class="margin2"></div>

Options

Option Data-* Default Description
steps data-steps 0 Steps for stepper. Int, serialized array or function name. Function must return a array of string with steps names.
states data-states null States for the steps of stepper. serialized array or function name. Function must return a array of string. States ca be: current, complete, warning, error or empty string
startStep data-start-step 1 Starting step of stepper
label data-label empty Step prefix if steps defined as Int
clickable data-clickable false
onStep data-on-step $.noop Callback to step change

Methods

Method Params Description
next Goto next step
prev
state step_index, state Goto prev step

Object

Core_CSS provides coreSteppers object to steppers manipulation. coreSteppers contains next methods:

  • coreSidebar.isStepper(el)
  • coreSidebar.next(el)
  • coreSidebar.prev(el)
  • coreSidebar.state(el, step, state)
