Options

Option Data-* Default Description
toggle data-toggle null Element to toggle sidebar state
shift data-shift null Elements with comma separated hwo can be shifted when sidebar is open
overlay data-overlay false Show or not overlay
duration data-duration 300 Speed to shifting elements

Methods

Method Params Description
open Open sidebar
close Close sidebar
toggle Toggle state of sidebar
isOpened Check if sidebar is opened

Object

Core_CSS provides coreSidebar object to sidebar manipulate. coreSidebar contains next methods:

  • coreSidebar.isSidebar(el)
  • coreSidebar.open(el)
  • coreSidebar.close(el)
  • coreSidebar.toggle(el)
