Customize

If you want to customize Core_CSS or add new widget or css class then you can add your own js or less files and run Grunt.

Before use Grunt you must install nodejs and last npm.

Grunt

Goto project directory: 
        cmd:\> cd path_to\Core_CSS
Install grunt: 
        cmd:\> npm install -g grunt-cli
Install project dependencies: 
        cmd:\> npm install
Add your files:
  • Widget JS - add js file to js/widgets
  • Utils JS - add js file to js/utils
  • Less (css) component - add less file to less/core and add import to core.less
  • Less (css) utils - add less file to less/utils and add import to core.less
  • Less (css) colors - add variables to variables.less file and create classes in core-colors.less
Run grunt: 
        cmd:\> grunt

To change grunt tasks you can modify Grunfile.js in project directory.
