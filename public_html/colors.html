Color classes

Colors classes defined in core.css (fg-* and bg-*) and core-colors.css (bd-*, ol-*, st-*, active and focus). To set color class to element you can use next classes (replace * with color name):

  • fg-* - text color
  • bg-* - background color
  • bd-* - border color
  • ol-* - outline color
  • st-* - stroke color
  • fg-*-active - text color for active element
  • bg-*-active - background color for active element
  • bd-*-active - border color for active element
  • ol-*-active - outline color for active element
  • fg-*-focus - text color for focused element
  • bg-*-focus - background color for focused element
  • bd-*-focus - border color for focused element
  • ol-*-focus - outline color for focused element
        <p class="fg-white bg-red">Lorem ipsum dolor mesum.</p>
        <button class="action-button fg-white bg-red bg-green-700-active">+</button>
