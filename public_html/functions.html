Utilities functions

Core_CSS provides nay utilities function stored in coreUtils object.

Functions

Name Description
isUrl(string) Check value is url
isColor(string) Check value is hex color
secondsToTime(num) Convert seconds to time. Return object {"h": hours, "m": minutes, "s": seconds}
hex2rgba(hex, alpha) Convert hex string to rgba value with alpha factor. Return value is string 'rgba(...)'
random(from, to) Generate random value from ... to
isInt(any) Check if value is a integer
isFloat(any) Check if value is a float
uniqueId(prefix) Generate unique ID in format xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx
isTouchDevice() Check if device is touchable
secondsToFormattedString(num) Convert seconds to HH:MI:SS
callback(cb, args) Cool for creating callbacks. cb - function name or real function or js code, args - arguments
exec(cb, args) Execute function. cb - function name or real function or js code, args - arguments
isFunc(func) Check if value is function
isCoreObject(el, role_name) Check if value is Core_CSS widget
addLocale(data) Add custom locale to coreLocales in runtime
getLocaleNames() Get Core_CSS locale names

Example

        coreUtils.isTouchDevice();
