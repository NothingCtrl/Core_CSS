Grids

Core_CSS provides two type of grid: grid and grid-list. Grid list build as flex model. Grid build as float model. 

        <div class="grid-list">
            <div class="row">
                <div class="cell"></div>
                ...
                <div class="cell"></div>
            </div>
        </div>

        <div class="grid">
            <div class="row [cells2...cells12]">
                <div class="cell [colspan2...colspan12] [offset2...offset12]"></div>
                ...
                <div class="cell"></div>
            </div>
        </div>
