Ripple effect

Core_CSS provides simple method to create ripple effect when object clicked. For this feature Core_CSS contains Ripple widget.

To add ripple effect for object you must add attribute data-role with value ripple and if need define ripple color over attribute data-ripple-color. Also you can define sub elements for ripple effect over attribute data-ripple-target. To define alpha factor for ripple effect, you can use attribute data-ripple-alpha.

Options

Option Data-* Default Description
rippleColor data-ripple-color #ffffff Ripple effect color
rippleAlpha data-ripple-alpha 0.4 Ripple alpha factor (transparency) for ripple color
rippleTarget data-ripple-target 0.4 Sub elements to make ripple effect
