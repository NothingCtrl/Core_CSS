Progress & Activity

Progress and activity indicators are visual indications of an app loading content.

Core_CSS provides any types of progress such as linear and circular.

Linear


        <div data-role="progress" data-value="35"></div>

        <div data-role="progress" data-type="line"></div>

        <div data-role="progress"
                data-type="buffered"
                data-bar-color="bg-cyan"
                data-buffer-color="bg-red"
                data-value="35"
                data-buffer="55"></div>

Circular


        <div data-role="progress"
                data-type="circle"></div>

        <div data-role="progress"
                data-type="circle"
                data-bar-color="st-yellow"></div>

        <div data-role="progress"
                data-type="circle"
                data-size="32"
                data-radius="10"
                data-bar-color="st-red"></div>

Options

Option Data-* Default Description
value data-value 0 Progress value
buffer data-buffer 0 Progress buffer value
type data-type default Progress type: default, line, circle, buffered
barColor data-bar-color class bg-green Progress bar color - hex value or class name
bufferColor data-buffer-color class bg-yellow Progress buffer color - hex value or class name
color data-color class bg-gray-600 Progress back color - hex value or class name
size data-size 64 Progress size for circle
radius data-radius 32 Progress bar size for circle
onChange data-on-change $.noop Callback for value change event
onEnd data-on-end $.noop Callback for value end

Methods

Method Params Description
val value New value for progress bar
buffer value New value for progress buffer

Dialog mode

Core_CSS provides method to create progress as dialog. To create progress as dialog you must use coreProgress or $.Progress object and call showPreloader method.

coreProgress

Method Params Description
isProgress element Check if element is progress
value el, value New value for progress bar
buffer el, value New value for progress buffer
showPreloader size, timeout Show circle preloader as dialog
