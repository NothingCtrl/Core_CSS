CoreCss

Установка

Репозиторий:

  • git@github.com:imena/core-css.git
  • https://github.com/imena/core-css.git

Скомпиленные файлы скриптов и стилей можно взять из папки на гитхабе.

CSS

Обязательное подключение 3х css-файлов: core.css, core-colors.css, core-icons.css


    <link rel="stylesheet" type="text/css" href="css/core.css">
    <link rel="stylesheet" type="text/css" href="css/core-colors.css">
    <link rel="stylesheet" type="text/css" href="css/core-icons.css">

JS

CoreCss базируется на jQuery.

Файлы всех компонентов собраны в core.js || core.min.js.


    <script src="js/jquery-3.0.0.min.js"></script>
    <script src="js/core.js"></script>
